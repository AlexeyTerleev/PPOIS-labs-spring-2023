
## **Для запуска программы пользователю предлагаю ввести несколько команд**:

    1. python3 main.py cli user_command - запускает симуляцию в коммандной строке
    2. python3 main.py gui - запускает симуляцию в грфическом окне


## **Для управление симуляцией в командной строке есть несколько команд**:

    добавить грядку - add.
    удалить грядку "номер грядки" - rm 4.
    удобрить грядку "номер грядки" - fert 4.
    полить грядку "номер грядки" - water 4.
    вылечить грядку "номер грядки" - hp 4.
    посадить "дерево/растение" "название растения" "название его плодов" "номер грядки" - new t/p name product_name 4.
    уничтожить вредителей "номер грядки" - rmpest 4.
    прополоть грядку "номер грядки" - weed 4.
    собрать урожай "номер грядки" - take 4.
    погода - weather.
    урожай - harvest.
    далее - next.
    показать огород - show.
