Для создания GUI была использована библиотека PyQt5.

Функции из меню:
    -Файл:
        -Сохранить - сохранение в файл табличных данных (при этом, если табличные данные сохраняются в первый раз и не
            были загружены из файла, то будет создан файл autosave{index}, где index будет вычислен автоматически)
        -Сохранить как - сохранение табличных данных в выбранный через диалоговое окно файл
        -Загрузить - загружает табличные данные из выбранного в диалоговом окне файла
    -Сортировка:
        Сортирует табличные данные по критериям:
            -Название (параметр по умолчанию) - по названию книг
            -Издательство - по названию издательства
            -Тираж - по тиражу
            -А-Я (параметр по умолчанию) - прямой порядок сортировки (a-z, а-я, 1-...)
            -Я-А - обратный порядок сортировки (z-a, а-я, ...-1)
    -Таблица:
        Отвечает за манипуляции над данными с помощью команд:
            -Поиск книг - вызывает диалоговое окно с таблицей и возможностью выбора критерия поиска (критерии описаны
                в условии лр). Данные в таблице могут быть изменены. После применения соответствующие данные в основной
                таблице изменяться.
            -Удаление книг - вызывает диалоговое окно для выбора критерия удаления (критерии описаны в условии лр)
            -Добавление книги - вызывает диалоговое окно, в котором описываются параметры добавляемой книги (книга будет
                добавлена в основную таблицу)

Непосредственно таблица:
    Данные в таблице могут быть изменены. При этом после нажатия любой из 5 нижних клавиш данные в таблице применяться
        и будет изменены. Данные выводяться по 10 элементов на странице.

Клавиши:
    -Крайняя левая (двойная левая стрелка) - осуществляет переход на первую страницу.
    -Средняя левая (одиночная левая стрелка) - осуществляет переход на предыдущую страницу.
    -Средняя (синхронизация) - обновляет данные на текущей странице.
    -Средняя правая (одиночная правая стрелка) - осуществляет переход на следущую страницу.
    -Крайняя правая (двойная правая стрелка) - осуществляет переход на последнюю страницу.
